# SIUP (Sistem Uploader)
help upload exam results during practice exams

## How to use 
- Clone this Repo
  ```
  https://github.com/Farriq-mfq/uploader-ujian.git
  ```
- Run command 
  ```
  composer install
  ```
  ```
  cp .env.example .env
  ```
  ```
  php artisan key:generate
  ```
## Fitur 
- Multiple rooms / exam uploader
- ftp connected 
- storage management
